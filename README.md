# BeeTSP: A Johnson-Compliant TSP Evaluation Protocol

Implements experimental standards from:
David S. Johnson. "A Theoretician's Guide to the Experimental Analysis of Algorithms" (2001).

## Usage
python -m scripts.run_johnson_protocol

this file reads *.json in the configs directory and executes the commands within.

python -u bee_tsp.titration.johnson_audit

python -u bee_tsp.titration.statistics
